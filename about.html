About This Project

This UI project is designed to remix and combine Bootstrap 5 components in a creative and modern way. The focus is on layout composition, clean UI, and responsive design, not just copy-pasting official examples but creating synergy between components.

  • Navbars, Cards, Carousels, Hero sections, Forms, and Modals are used
  • All code is well-indented and readable
  • Deployed using GitHub Pages / Netlify
